# M19.1 Overview

This package introduces the first step of the separate content pipeline.

## Included
- `GameProject/` - the current playable game project
- `LevelBakerProject/` - a separate Godot project for future baking work
- `SharedLevelContent/` - shared folders for source images, baked levels, previews, and the canonical baked level definition script

## M19.1 scope
- separate baker project exists
- shared baked level format exists
- baker UI shell exists
- validation/sample resource workflow exists
- actual image baking/export logic is intentionally deferred to M19.2
