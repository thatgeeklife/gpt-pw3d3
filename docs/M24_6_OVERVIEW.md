# M24.6 Overview

M24.6 is a theme authoring reliability pass built on M24.5.3.

The goal is to make imported model and wrapper state visible instead of mysterious. The validation report now includes an Asset Health section for every scene slot, and each scene row has a Rebuild Wrapper button.

## What changed

- Added per-field `Rebuild Wrapper` buttons.
- Added detailed `Asset Health` output to the validation report.
- Added stronger metadata to newly generated wrapper scenes.
- Cleaned up save/export/validation status messages.
- Preserved managed import folders in the zip with README files.

## What did not change

M24.6 does not change Steam networking, player spawning, portal flow, holding room behavior, level transitions, or runtime lobby logic.
