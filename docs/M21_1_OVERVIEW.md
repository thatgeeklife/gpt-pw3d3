# M21.1 Overview

This milestone applies the first world-scale pass for the modular theme roadmap.

## Main change
The gameplay grid stays the same, but each logical gameplay cell now renders at roughly **2x** the previous world-space width/depth.

## Included in M21.1
- larger cell/world size contract
- level generator floor + wall sizing updated to use the larger world size
- runtime cell-to-world conversion updated
- tile footprint visuals increased
- highlight and orb visuals increased
- player movement / zoom tuned for the larger world scale
- tile interaction distances and label radius retuned

## Not yet in M21.1
- modular floor piece sets
- weighted wall variants
- theme-authored wall categories
- asset-normalized modular kit import
